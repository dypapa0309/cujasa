# CUJASA 고객용 간단 사용 매뉴얼

CUJASA는 쿠팡 파트너스용 Threads 자동 포스팅 솔루션입니다. 고객은 처음에 로그인, Threads 연결, 쿠팡 파트너스 API 설정, 콘텐츠 기준 저장만 해두면 이후 `자동화 시작`으로 예약 포스팅을 만들고 운영 상태를 확인할 수 있습니다.

## 1. 로그인부터 헷갈리지 않기

CUJASA 로그인 화면에는 `로그인`과 `회원가입`이 함께 있습니다.

### 이미 계정을 받은 고객

1. 접속 주소로 들어갑니다.
2. 화면 오른쪽의 제목이 `워크스페이스 로그인`인지 확인합니다.
3. `아이디 또는 이메일`에 담당자가 안내한 아이디 또는 이메일을 입력합니다.
4. `비밀번호`를 입력합니다.
5. `로그인`을 누릅니다.

주의할 점:

- 이메일 계정만 받았으면 이메일을 입력합니다.
- 별도 아이디를 만들었거나 안내받았으면 아이디를 입력해도 됩니다.
- `회원가입`은 새 계정을 만드는 버튼입니다. 이미 계정을 받은 고객은 누르지 않아도 됩니다.

### 처음 직접 가입하는 고객

1. 로그인 박스 오른쪽 위의 `회원가입`을 누릅니다.
2. `고객명`과 `연락처`를 입력합니다.
3. `사용할 솔루션`에서 `CUJASA · 쿠팡 파트너스 자동화`를 선택합니다.
4. 사용할 `아이디`를 입력합니다. 영문/숫자 3자 이상으로 만듭니다.
5. `비밀번호`와 `비밀번호 확인`을 입력합니다.
6. 개인정보 수집 및 이용 동의에 체크합니다.
7. `무료로 시작하기`를 누릅니다.

가입 후에는 자동으로 워크스페이스 화면으로 이동합니다.

### 로그인이 안 될 때 먼저 확인할 것

- 로그인 화면이 `회원가입` 상태라면 `로그인` 버튼을 눌러 `워크스페이스 로그인` 화면으로 바꿉니다.
- 아이디로 가입했다면 이메일 대신 아이디를 입력합니다.
- 이메일로 안내받았다면 이메일 전체 주소를 입력합니다.
- 비밀번호는 대소문자를 구분합니다.
- 그래도 안 되면 담당자에게 고객명, 아이디 또는 이메일, 보이는 오류 문구를 보내주세요.

## 2. 첫 화면에서 보는 것

로그인하면 고객용 워크스페이스로 들어갑니다. 기본 화면은 `워크스페이스`이며 CUJASA 작업은 보통 이 화면에서 시작합니다.

주요 메뉴:

- `워크스페이스`: 솔루션 선택과 주요 작업 버튼이 모여 있는 화면입니다.
- `자동화 실행`: 설정 점검 후 자동화를 켜거나 끄는 화면입니다.
- `포스팅 현황`: 예약, 완료, 실패, 확인 필요 포스팅을 보는 화면입니다.
- `결제`: 이용권과 결제 상태를 확인하는 화면입니다.
- `설정`: Threads 연결, 쿠팡 파트너스 API, 콘텐츠 기준, 업로드 수를 설정하는 화면입니다.

## 3. Threads 계정 준비

CUJASA는 Threads에 글을 올리는 자동화이므로 Threads 계정 연결이 필요합니다.

1. `설정`으로 이동합니다.
2. `계정 기본 정보`에서 `Threads 연결` 상태를 확인합니다.
3. `Threads 연결하기`를 누릅니다.
4. 안내창이 나오면 아래를 먼저 확인합니다.
5. Chrome 또는 Safari에서 `threads.net`에 올바른 Threads 계정으로 로그인합니다.
6. Threads 웹에서 CUJASA 연결 승인을 진행합니다.
7. 승인 후 CUJASA로 돌아와 연결을 완료합니다.

고객이 자주 헷갈리는 점:

- Threads 앱에만 로그인되어 있으면 연결이 실패할 수 있습니다.
- 브라우저의 `threads.net` 로그인 계정이 CUJASA에 연결하려는 계정과 같아야 합니다.
- 다른 Threads 계정이 뜨면 브라우저에서 로그아웃 후 올바른 계정으로 다시 로그인합니다.
- 연결 상태가 `토큰 갱신 실패`로 보이면 `다시 연결하기`를 눌러 재연결합니다.

## 4. 쿠팡 파트너스 API 설정

CUJASA는 실제 쿠팡 상품과 파트너스 링크가 매칭된 글만 예약합니다.

1. `설정`으로 이동합니다.
2. `쿠팡 파트너스 API 설정`을 엽니다.
3. `Access Key`, `Secret Key`, `Partner ID`를 입력합니다.
4. 계정별 추적을 따로 하고 싶으면 `Tracking Code`도 입력합니다.
5. 아래의 `설정 저장`을 누릅니다.

주의할 점:

- 이미 저장된 값은 보안상 그대로 보이지 않을 수 있습니다.
- placeholder에 `저장됨 - 변경 시에만 입력`이 보이면 기존 값이 저장되어 있다는 뜻입니다.
- 빈칸으로 두면 기존 저장값은 유지됩니다.
- 쿠팡 상품 매칭이 안 되는 날은 품질 보호를 위해 포스팅을 예약하지 않을 수 있습니다.

## 5. 콘텐츠 기준 설정

자동화 품질은 `설정`의 콘텐츠 기준에 가장 크게 영향을 받습니다.

필수로 확인할 항목:

- `타겟층`: 누구에게 말할지 적습니다. 예: 자취하는 직장인, 30대 주부
- `다룰 카테고리`: 어떤 상품군을 다룰지 적습니다. 예: 주방용품, 청소, 수납
- `콘텐츠 방식`: 공감형, 질문형 등 글의 방향을 선택합니다.
- `강도`: 표현의 세기를 선택합니다.
- `댓글 유도`: 댓글을 어떻게 유도할지 선택합니다.
- `상품 언급`: 상품을 자연스럽게 언급할지, 직접적으로 언급할지 선택합니다.
- `이모지`: 이모지 사용 정도를 선택합니다.
- `추가 요청사항`: 말투나 운영 원칙을 적습니다.
- `다루지 말 것`: 피해야 할 주제를 줄바꿈으로 입력합니다.
- `금지어`: 쓰면 안 되는 표현을 줄바꿈으로 입력합니다.

입력 후 반드시 `설정 저장`을 누릅니다.

## 6. 포스팅 스케줄 설정

1. `설정`에서 `포스팅 스케줄`을 엽니다.
2. `하루 최대 포스팅`을 입력합니다.
3. `분산 기준 시각`을 선택합니다.
4. `설정 저장`을 누릅니다.

현재 고객 화면 기준으로 하루 최대 포스팅은 최대 5개입니다. 이 숫자는 보장 발행 수가 아니라 상한입니다. 실제로는 상품 매칭과 링크 생성이 성공한 글만 예약됩니다.

## 7. 자동화 시작하기

1. `자동화 실행`으로 이동합니다.
2. 먼저 `사전 점검`을 누릅니다.
3. `실행 가능` 또는 `현재 설정 점검을 통과했습니다`가 보이면 준비가 된 상태입니다.
4. `자동화 시작`을 누릅니다.
5. 예약 작업이 시작되면 오늘 예약을 만들고, 이후 매일 설정값 기준으로 계속 운영됩니다.

자동화 시작 전에 막히는 경우:

- Threads 연결이 안 되어 있거나 토큰이 만료된 경우
- 쿠팡 파트너스 API 값이 없거나 잘못된 경우
- 콘텐츠 기준이 비어 있는 경우
- 무료 사용 기간이 종료된 경우

이 경우 화면의 안내를 보고 `설정` 또는 `결제`로 이동해 필요한 항목을 처리합니다.

## 8. 자동화 중지하기

1. `자동화 실행`으로 이동합니다.
2. 상태가 `자동화 진행 중`인지 확인합니다.
3. `자동화 중지`를 누릅니다.

중지하면 새 예약 생성이 멈춥니다. 이미 만들어진 기존 예약과 게시 기록은 `포스팅 현황`에서 확인할 수 있습니다.

## 9. 포스팅 현황 확인하기

`포스팅 현황`에서는 예약과 결과를 확인합니다.

상태별 의미:

- `예약`: 정해진 시간에 Threads에 업로드될 예정입니다.
- `완료`: Threads 업로드가 끝난 글입니다.
- `확인 필요`: 실패, 댓글 링크 문제, 수동 확인이 필요한 글입니다.
- `상품 매칭 누락`: 글은 있어도 수익화 가능한 쿠팡 상품 링크가 부족한 상태입니다.

포스팅을 눌러 상세를 열면 글 내용, 연결 상품, 오류 안내, Threads 링크를 확인할 수 있습니다.

## 10. 홈에서 성과 보기

`홈`에서는 계정의 전체 상태를 빠르게 확인합니다.

- 자동화가 실행 중인지 또는 중지되어 있는지
- 예약 대기 개수
- 총 포스팅 수
- 총 클릭 수
- 다가오는 예약 포스팅
- 최근 포스팅

클릭이 잘 나오는 주제는 `설정`의 다룰 카테고리나 추가 요청사항에 더 구체적으로 반영하면 좋습니다.

## 11. 결제와 이용 상태

`결제` 메뉴에서 이용권 상태를 확인합니다.

- 무료 사용이 종료되면 자동화 실행이 막힐 수 있습니다.
- `이용 기간 만료`가 보이면 결제 또는 연장이 필요합니다.
- 결제 관련 문의는 담당자에게 고객명과 계정 아이디를 함께 전달합니다.

## 12. 문제 발생 시 담당자에게 보낼 정보

문제가 생기면 아래 정보를 같이 보내면 확인이 빠릅니다.

- 고객명
- 로그인 아이디 또는 이메일
- CUJASA 계정 이름
- Threads 핸들
- 문제가 발생한 메뉴: 로그인, 설정, 자동화 실행, 포스팅 현황, 결제
- 화면에 보이는 오류 문구
- 가능하면 화면 캡처

## 빠른 시작 요약

1. `워크스페이스 로그인`에서 아이디 또는 이메일로 로그인합니다.
2. `설정`에서 Threads를 연결합니다.
3. `설정`에서 쿠팡 파트너스 API 값을 저장합니다.
4. `설정`에서 타겟층, 카테고리, 말투, 금지 주제를 저장합니다.
5. `설정`에서 하루 최대 포스팅과 분산 기준 시각을 저장합니다.
6. `자동화 실행`에서 `사전 점검`을 누릅니다.
7. 점검을 통과하면 `자동화 시작`을 누릅니다.
8. `포스팅 현황`과 `홈`에서 예약, 완료, 클릭 수를 확인합니다.
