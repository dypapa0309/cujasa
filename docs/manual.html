CUJASA 사용 메뉴얼

쿠팡 파트너스 자동화 솔루션 CUJASA를 처음 사용하시는 분을 위한 가이드입니다.
설정 → 자동화 실행 → 수익 확인까지 순서대로 따라오시면 됩니다.

CUJASA가 하는 일

CUJASA는 쿠팡 파트너스 수익을 자동으로 만들어 주는 솔루션입니다. 아래 4가지 작업을 사람 손 없이 자동으로 처리합니다.

🤖
AI 주제 생성
타겟층에 맞는 Threads 포스팅 주제를 AI가 자동 생성
🛍️
쿠팡 상품 검색
주제와 관련된 쿠팡 상품을 API로 자동 검색 및 링크 삽입
✍️
콘텐츠 자동 작성
설정한 톤·타겟에 맞는 Threads 글을 자동으로 작성
📅
자동 예약 발행
설정한 시간대에 맞춰 Threads에 자동으로 업로드
💡 처음 시작 순서
  1. 로그인
  2. Threads 계정 추가
  3. 계정 설정 (타겟층, 주제 범위, 시간대 등)
  4. "저장하고 시작하기" 버튼으로 자동화 실행
  5. 포스팅 현황에서 결과 확인

Threads 계정 추가

CUJASA는 여러 Threads 계정을 관리할 수 있습니다. 구매 플랜에 따라 최대 계정 수가 달라집니다 (기본 플랜: 최대 4개).

계정 추가 방법

1
상단 탭에서 "+ 계정 추가" 버튼 클릭
화면 상단에 현재 등록된 계정 탭들이 보이고, 마지막에 추가 버튼이 있습니다.
2
계정 이름과 Threads 아이디 입력
계정 이름은 내부 관리용 이름입니다 (예: "자취꿀템 계정").
Threads 핸들은 @ 없이 입력합니다 (예: myaccount_threads).
3
확인 버튼 클릭
계정이 생성되면 자동으로 해당 계정의 설정 화면으로 이동합니다.
✅ 계정 이름 예시
  • • 자취꿀템 계정
  • • 육아 계정
  • • 주방용품 계정
  • • 직장인 계정
⚠️ 계정 수 제한 도달 시

플랜에서 허용하는 최대 계정 수에 도달하면 추가 버튼이 비활성화됩니다. 계정을 더 사용하려면 담당자에게 문의주세요.

계정 설정

각 계정마다 AI가 어떤 콘텐츠를 만들지 설정합니다. 이 설정이 자동화의 핵심입니다.

A 기본 정보
계정 이름
내부 관리용 이름. 예: "자취꿀템 계정"
Threads 핸들
실제 Threads 계정의 @ 아이디. 예: myhandle
B 콘텐츠 설정 (가장 중요!)
*필수 타겟층 (Target Audience)
어떤 사람들을 위한 콘텐츠를 만들지 설명합니다. 구체적일수록 좋습니다.
예시
30대 주부, 자취하는 직장인, 신혼부부
*필수 주제 범위 (Content Scope)

예시
주방용품, 청소용품, 수납, 생활잡화
톤 / 말투 (Tone)

예시

💡 설정 팁

타겟층과 주제 범위를 구체적으로 적을수록 AI가 더 관련성 높은 글을 생성합니다. 처음에는 1~2가지 카테고리로 시작해 점차 늘리는 것을 권장합니다.

자동화 시작하기

설정을 완료했다면 "저장하고 시작하기" 버튼으로 자동화를 실행합니다. 이 버튼 하나로 4가지 과정이 자동으로 진행됩니다.

주제 생성

↓

쿠팡 상품 검색

↓
3
콘텐츠 작성
설정한 톤과 타겟에 맞는 Threads 포스팅 글이 자동으로 완성됩니다
↓
4
예약 큐 등록
완성된 글이 설정한 시간대에 자동 발행되도록 큐에 등록됩니다

포스팅 현황

생성된 모든 포스팅의 상태를 확인하고 관리하는 화면입니다. 하단 내비게이션의 "포스팅 현황" 탭을 누르면 진입합니다.

포스팅 상태 구분

예약됨
발행 대기 중인 포스팅. 설정한 시간대에 자동으로 Threads에 업로드됩니다.
완료
Threads에 성공적으로 업로드된 포스팅. 링크를 눌러 직접 확인할 수 있습니다.
확인 필요
실패하거나 문제가 생긴 포스팅. 담당자에게 문의하거나 재시도할 수 있습니다.
포스팅 상세 보기

포스팅 항목을 클릭하면 아래 정보가 펼쳐집니다:

•글 내용 — AI가 작성한 전체 포스팅 텍스트
•연결 상품 — 포스팅과 함께 연결된 쿠팡 상품 이미지, 이름, 가격
•클릭 수 — 완료된 포스팅의 경우 링크 클릭 수 표시
•Threads 링크 — 완료된 포스팅에서 직접 Threads 게시물로 이동
💡 포스팅은 언제 올라가나요?

자동화 실행 후 생성된 포스팅은 계정 설정의 "활성 시간대" 내에서 랜덤하게 분산 발행됩니다. 예를 들어 08:00~22:00으로 설정하고 하루 5개 발행이면, 해당 시간대 안에서 균등하게 나뉘어 자동 업로드됩니다.

자주 묻는 질문

Q. 쿠팡 파트너스 계정이 없어도 CUJASA를 쓸 수 있나요?
쿠팡 파트너스 계정은 반드시 필요합니다. CUJASA는 쿠팡 파트너스 API를 통해 상품을 검색하고 추적 링크를 자동 생성합니다. 파트너스 계정 가입은 쿠팡 파트너스 사이트에서 무료로 할 수 있습니다.
Q. Threads 계정이 없는데 어떻게 해야 하나요?
Threads 계정이 반드시 필요합니다. Instagram 계정으로 쉽게 만들 수 있으며, 팔로워가 0명인 신규 계정도 바로 시작할 수 있습니다. 계정 연동은 담당자가 설정 과정에서 함께 도와드립니다.
Q. 자동화를 한 번 실행하면 계속 돌아가나요?
"저장하고 시작하기"를 누르면 그 시점에 콘텐츠를 생성하고 큐에 등록합니다. 큐에 등록된 포스팅은 설정한 시간대에 자동으로 발행됩니다. 큐가 소진되면 다시 "저장하고 시작하기"를 눌러 새 콘텐츠를 채워주세요.
Q. 생성된 글이 마음에 들지 않으면 어떻게 하나요?
계정 설정의 타겟층, 주제 범위, 톤/말투를 더 구체적으로 수정한 뒤 다시 실행해 보세요. 또한 마음에 드는 글의 스타일을 "CTA 방식" 또는 "톤" 항목에 추가로 작성하면 AI가 그 방향으로 글을 생성합니다.
Q. 포스팅이 실패했을 때 어떻게 하나요?
"포스팅 현황"에서 "확인 필요" 상태의 포스팅을 확인하세요. 일시적인 오류라면 재시도가 자동으로 이루어집니다. 지속적으로 실패한다면 담당자에게 오류 내용을 캡처해서 문의해 주세요.
Q. 환불 정책이 어떻게 되나요?
결제 후 7일 이내 환불 신청 시 구매 가격의 20%가 환불됩니다. 7일 이후에는 환불이 불가합니다. 담당자(010-4094-1666)에게 문의해 주세요.
Q. 계정을 4개보다 더 사용하고 싶어요.
기본 플랜은 최대 4개 계정을 지원합니다. 더 많은 계정이 필요하시면 담당자에게 별도 문의해 주세요.
